Welcome to the Rent/Send API Documentation!

The Rent/Send API is a powerful tool that allows developers to integrate the functionality of renting rooms, sending money, and managing user information seamlessly into their applications. Whether you're building a financial app, a room-sharing platform, or any other service that requires secure and efficient room rental, money transfer, user management, and eLoad capabilities, our API has you covered. Our API offers a comprehensive set of endpoints and resources that enable you to interact with the Rent/Send system programmatically. With just a few simple requests, you can access features like room search, booking management, payment processing, user information updates, and eLoad transactions, all while ensuring the privacy and security of your users' data.

Key Features:

  • Room Rental: Search for available rooms, retrieve room details, and manage bookings effortlessly.
  • Money Transfer: Facilitate secure and fast money transfers between users, streamlining the payment process.
  • User Management: Allow users to update their information, such as their first name and other relevant details, using their unique user ID.
  • eLoad System: Enable users to perform electronic load transactions, such as mobile phone top-ups and prepaid credits, seamlessly within your application.

Getting Started:

To start using the Rent/Send API, you'll need an API key, which you can obtain by signing up for an account on our developer portal. Once you have your API key, you can authenticate your requests by including it in the request headers.

API Resources:

Our API provides several resources that allow you to perform specific actions. These include:

  • Rooms: Interact with room listings, availability, and booking information.
  • Users: Manage user profiles, authentication, and access control. Users can update their information using their unique user ID.
  • Payments: Handle payment processing and transactions securely.
  • Transfers: Initiate and track money transfers between users.
  • eLoad: Perform electronic load transactions, such as mobile phone top-ups and prepaid credits.

Please refer to the API documentation for detailed information on endpoints, request/response formats, authentication methods, and example use cases. We also provide client libraries and code snippets in various programming languages to help you get started quickly.
